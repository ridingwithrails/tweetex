## Installation
Build:
```
  ./build.sh
```

Run:

### Add an .env file with the following 

```
# .env

TWTAPI_CONSUMER_KEY=KEY
TWTAPI_CONSUMER_SECRET=SECRET
TWTAPI_ACCESS_TOKEN=TOKEN
TWTAPI_ACCESS_TOKEN_SECRET=TOKEN_SECRET
```

Then....

```
./run.sh
```
